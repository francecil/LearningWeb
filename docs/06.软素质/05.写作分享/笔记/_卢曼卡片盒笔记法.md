1. 笔记原子化，每条笔记只记录一个想法，用自己的话记
2. 给每条笔记分配唯一标识符（唯一的超链接）
3. 在笔记中引用其他笔记，形成网络
4. 建立结构化，帮助组织和导航（这里的难点是结构化怎么建，结构化到什么程度？）

总结：笔记全铺平，笔记原子化，双链笔记，允许多分类
