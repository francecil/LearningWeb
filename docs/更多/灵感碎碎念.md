记录一闪而过的灵感，比如立项思路、文章主题、解决方案，亦或只是一个信息标记。

记录采用 [GitHub Issue Comment](https://github.com/francecil-public/blog-comments-utterances/issues/6) 持久化，即本文下方的评论形式。

记录方式有两种：
- 博客评论（链路较长）
- 微信消息 + GitHub 同步（方便快速）

灵感评论会定期维护（暂定一个月一次），若灵感已完成（🎉）或放弃（😕），用表情标注，也可以选择删除
